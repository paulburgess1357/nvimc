# Commands

See [KEYBINDINGS.md](KEYBINDINGS.md) for key bindings.

## General

| Command | Action |
| ------- | ------ |
| `:Files` | Find files |
| `:Buffers` | Buffer list |
| `:Rg` / `:Grep` | Live grep |
| `:Symbols` | Document symbols |
| `:Marks` | Marks |
| `:Help` | Help tags |
| `:Commands` | Commands |
| `:Keymaps` | Keymaps |
| `:Aerials` | Toggle outline sidebar |

## LSP

| Command | Action |
| ------- | ------ |
| `:LspIndexAll` | Force LSP to index all project files |
